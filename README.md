# 🌿 AYURWELL -- Ayurvedic Healthcare Platform

AYURWELL is a modern web-based Ayurvedic healthcare platform built using
React, Vite, and Tailwind CSS.\
It provides features like doctor-patient login, appointment booking,
dosha quiz, and product purchasing.

------------------------------------------------------------------------

## 🚀 Tech Stack

-   React.js
-   Vite
-   Tailwind CSS
-   Node.js

------------------------------------------------------------------------

## 📦 Installation & Setup

### 1️⃣ Clone the repository

git clone https://github.com/your-username/your-repo-name.git cd
your-repo-name

### 2️⃣ Install dependencies

npm install

### 3️⃣ Run development server

npm run dev

App will run at:

http://localhost:5173/
